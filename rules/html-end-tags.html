# mpx/html-end-tags

强制结束标记样式

  • ⚙️ 这条规则包含在"plugin:mpx/mpx-essential"。
  • 🔧 --fix选项命令行 (opens new window)可以自动修复此规则报告的一些问题。

# 📖 规则详情

该规则旨在禁止缺少结束标记。
