# 介绍

Mpx的官方ESLint插件

这个插件允许我们检查.mpx文件中的<template>,<script>模块。

  • 查找语法错误
  • 发现错误使用 Mpx.js 模版指令 (opens new window)。

ESLint编辑器集成对于实时检查代码非常有用。
